心理学专业Linux应用清单
  • June 22, 2019
  • Application  
  • Application  
  • Software  
以下应用（软件）可在deepin操作系统上使用。其中大部分应用可从深度商店安装，但深度商店中的应用倾向于保守、版本普遍较旧。最新版应用可从相应网站下载、安装，但贸然使用最新版应用可能有导致系统崩溃的风险。 心理学专业 数据处理 EpiData（完善地数据录入功能，简单的统计功能，deepin上对中文的支持不友好） RevMan 5（元分析软件，依赖于java1.6或1.7） R（建议自己编译安装最新版） RStudio（使用dpkg命令安装最新版deb包） Mplus（使用PlayOnLinux安装windows版，也可直接使用Linux版） SPSS 24（提取码:hlep） JASP（类似SPSS，统计功能较少） 实验编程 PsychoPy（配合Anaconda使用） Python（deepin已经安装了Python，需要自己找一个顺手的IDE；亦可配合Anaconda使用） 文本编辑器 深度编辑器（支持代码语法高亮） 办公 Office WPS LibreOffice（可以直接导出矢量图） pdf编辑器 PDFsam basic（页面剪接、排序） 阅读 Mendeley(文献管理) 福昕阅读器（pdf文档，可添加多种批注） caj阅读器（caj文档，wine版，可从应用商店安装） 文档查看器（Evince，系统自带，可查看多种格式的文档） calibre（多种格式电子书的管理、阅读、格式转换） 压缩 深度管理器（自带压缩功能） 360压缩（应用商店安装，或使用命令安装sudo apt install 360zip） 输入法 中州韵（可从deepin论坛搜索安装方法） 浏览器 Chrome 360浏览器 图像 GIMP（媲美Photoshop） 深度截图（CTRL+ALT+A） 深度看图 视频 深度录屏（适合录制短时小视频，无语音） OBS Studio（适合录制课程视频） 深度影院（本地播放） 音乐播放 网易云音乐（在线，本地） 深度音乐（本地） 下载 Motrix（替代迅雷，支持断点续传，适合下载大文件） 网盘 百度网盘 即时通讯 米聊 Skype 微信（wine版，从应用商店安装） QQ，Tim（wine版，从应用商店安装） 邮件客户端 雷鸟客户端（从应用商店安装） 科学上网（需自有shadowsocks帐号） slectron-ssr（从应用商店安装） shadowsocks-deepin wine 有些狭隘的软件有windows版，但没有Linux版，可使用wine安装该类软件，但未必可用。
