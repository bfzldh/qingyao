tssem1 function in metaSEM package uses a list data, which includes a “son” list of covariance matrix and a vector of sample size. This data format differs a lot from the wide format we used usually. In the wide format data, one row includes information for one study, hence I save the correlations in a matrix (usually one matrix for one study) successively in a row. In the manual of metaSEM package, Cheung calls this matrix in a row “stacked matrix”.
